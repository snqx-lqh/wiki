    Swift 语言学习笔记

    Swift 语言简介与特点¶

    Swift 是苹果推出的一种现代编程语言，用于开发 macOS、iOS 等平台的软件。它强调**类型安全**和易读性，通过避免常见错误（如未初始化变量）来提高代码质量。Swift 编译器优化了执行效率，使程序运行更快；语言本身兼具脚本语言的**交互性**，支持直接在 REPL（或 Xcode Playground）中运行代码查看结果。总体来说，Swift 具有以下特点：

    • 安全：严格的类型系统和可选类型设计，减少运行时错误。

    • 快速：优化的编译器和语言特性让代码执行高效。

    • 现代：支持可选类型、闭包、泛型等现代编程范式，使代码更简洁灵活。

    • 交互性：可以使用 Xcode Playground 或命令行 REPL 实时运行 Swift 代码，便于学习和调试。

    开发环境搭建（macOS）¶

    在 macOS 上学习 Swift，通常有两种主要方式：Xcode IDE 或 命令行工具。最简单的方法是从 Mac App Store 下载并安装最新版本的 Xcode，它自带最新版的 Swift 编译器。例如，安装 Xcode 即可获得 Swift 6.0 编译器和相关开发工具。安装完成后，可在 Xcode 中新建 Swift 项目，或直接使用自带的 Swift REPL。

    另一个选择是安装 Swift 官方提供的独立工具链（.pkg 文件），无需安装整个 Xcode。这种方法会在系统中增加 swift 命令行环境，可以在终端中直接编写和运行 Swift 代码。此外，也可通过包管理工具（如 Homebrew）安装 Swift，但官方推荐使用上述两种方式。安装完成后，可以通过在终端输入 swift --version 来确认 Swift 版本，并使用 swift 命令启动 REPL 或使用 swiftc 编译器编译 .swift 文件。

    Swift 基础语法¶

    变量与常量¶

    Swift 使用 let 关键字声明**常量**（不可变）和 var 声明**变量**（可变）。例如：

    let maximumNumberOfLoginAttempts = 10
var currentLoginAttempt = 0

    以上代码中，maximumNumberOfLoginAttempts 声明为常量并赋值为 10，其值在之后无法更改；currentLoginAttempt 是变量，可随时修改。常量能让代码更安全，原则是“能用常量就用常量”。如果需要同时声明多个变量或常量，可用逗号分隔：

    var x = 0.0, y = 0.0, z = 0.0

    当需要明确类型时，还可以加上类型注解，例如：

    var welcomeMessage: String
welcomeMessage = "Hello"

    Swift 在大多数情况下支持类型推断，因此如果在声明时赋了初值，通常不用手动指定类型。

    数据类型¶

    Swift 支持多种基本数据类型：

    • 整型：Int，用于表示整数值。

    • 浮点型：Double（双精度）和 Float（单精度），用于表示小数值。

    • 布尔型：Bool，表示 true 或 false。

    • 字符串：String，用于文本数据。

    例如：

    let maxScore: Int = 100
var pi: Double = 3.14159
var isActive: Bool = true
var username: String = "Alice"

    Swift 还有三种常用的集合类型：Array（数组）、Dictionary（字典）和 Set（集合），用于存储有序列表、键值对和无序不重复元素。例如：

    var scores: [Int] = [100, 90, 80]
var userInfo: [String: Any] = ["name": "Alice", "age": 25]

    控制流¶

    Swift 提供了常见的流程控制结构：

    • 条件语句：if-else 和 guard（条件守卫）用于根据条件执行不同代码块。

    • switch 语句：比 C 语言的 switch 更强大，可匹配多种模式。

    • 循环：for-in 循环用于遍历数组、字典、范围（如 1...5）等可迭代对象；while 和 repeat-while 循环用于满足条件时重复执行代码。

    例如，下面用 if-else 判断成绩：

    let score = 85
if score >= 60 {
    print("Pass")
} else {
    print("Fail")
}

    用 for-in 循环遍历数字：

    for i in 1...3 {
    print(i)
}
// 输出：1 2 3

    if-else 和 for-in 示例展示了基本用法；更复杂的 switch、guard 和 while 用法可以在后续章节中学习。

    函数¶

    函数（Function）是封装了特定任务的代码块，用 func 关键字定义。一个函数可以有参数（输入）和返回值。例如：

    func greet(person: String) -> String {
    let greeting = "Hello, " + person + "!"
    return greeting
}
print(greet(person: "Anna"))  // 输出 "Hello, Anna!"

    上面定义了一个函数 greet(person:)，接收一个 String 参数并返回一段问候语。调用时使用 greet(person: "Anna")，函数返回字符串并输出。在 Swift 中，函数签名以参数类型和返回类型共同组成。函数可以没有返回值（即返回 Void），也可以没有参数。支持**参数标签**和**默认值**等高级特性。更多函数用法会在进阶学习中深入介绍。

    结构体与类¶

    结构体（struct）和类（class）是 Swift 中的两种复合类型，都可以定义属性和方法来组织数据和行为。结构体是值类型，类是引用类型，二者语法类似。下面示例定义了一个结构体 Resolution 和一个类 VideoMode：

    struct Resolution {
    var width = 0
    var height = 0
}
class VideoMode {
    var resolution = Resolution()
    var name: String?
}

    Resolution 结构体包含两个整数属性；VideoMode 类包含一个 Resolution 实例和一个可选字符串属性 name。使用时可以创建它们的实例：

    let res = Resolution(width: 1920, height: 1080)
let vm = VideoMode()
vm.resolution = res
print(vm.resolution.width)   // 输出 1920

    以上示例展示了如何定义和使用结构体与类。后续可以学习它们的初始化方法、继承、多态等面向对象特性。

    可选类型¶

    Swift 的**可选类型**用于表示变量可能有值也可能没有值（即 nil）。声明可选类型时在类型后加 ?，例如 Int? 表示“可选的整数”。例如：

    let possibleNumber = "123"
let convertedNumber = Int(possibleNumber)  // convertedNumber 的类型为 Int?

    上面 Int(possibleNumber) 返回 Int?，因为转换可能失败。可选值要么包含所需类型的值，要么为 nil。使用可选类型时，不能直接使用包含值的可选，需要解包才能获取内部值。常用的**可选绑定**语法如下：

    if let number = convertedNumber {
    print("\(number) is a valid integer")
} else {
    print("Conversion failed")
}

    在 if let 中，如果 convertedNumber 不为 nil，就将其值赋给新的常量 number 并进入分支，否则执行 else。这样可以安全地使用可选值。注意，! 操作符可以强制解包可选（如 convertedNumber!），但只有在非常确定值不为 nil 时才使用，以避免运行时错误。总体来说，可选类型增强了代码的安全性，需要处理 nil 情况才能访问值。

    学习建议与路径¶

    对零基础初学者而言，建议循序渐进地掌握 Swift 语言：

    • 基础语法：首先熟悉基本语法（变量/常量声明、数据类型、运算符、流程控制等）；

    • 函数与面向对象：接着学习函数定义与调用，理解结构体、类和继承等面向对象概念；

    • 集合与其他特性：掌握常用集合类型（数组、字典、集合）、错误处理、协议、泛型等高级概念。

    同时，要多**动手练习**：在 Xcode 或终端的 Swift REPL 中反复编写和运行代码示例，加深理解。可以参考苹果官方的《Swift 编程语言》教程、社区教程或优秀书籍，按部就班地系统学习。坚持练习，遇到问题参考文档和社区资源，逐步深化对 Swift 语言的掌握。

    参考资料： Apple 官方 Swift 语言文档、技术博客。

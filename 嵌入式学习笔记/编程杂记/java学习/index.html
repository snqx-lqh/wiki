    1、抽象类

    在父类中声明但是未定义

    //Three_dimension.java
public abstract class Three_dimension {
    double r;
    double h;
    Three_dimension(double radius,double height){
        r=radius;
        h=height;
    }
    public abstract double superficial_area();
    public abstract double volume();
}

//Circular_cylinder.java
public class Circular_cylinder extends Three_dimension{
    Circular_cylinder(double r,double h)
    {
        super(r,h);
    }
    public double superficial_area()
    {
        return r*r*Math.PI*2+r*h*Math.PI*2;
    }
    public double volume(){
        return r*r*Math.PI*h;
    }
}

//Circular_cone.java
public class Circular_cone extends Three_dimension {
    Circular_cone(double r,double h){
        super(r,h);
    }
    public double superficial_area(){
        return Math.PI*r*Math.sqrt(r*r+h*h);//圆椎体的表面积
    }
    public double volume(){
        return r*r*Math.PI*h/3;//圆椎体的体积
    }
}
//主类
//Example1.java
public class Example1 {

        Circular_cylinder c1 = new Circular_cylinder(2,10);
        Circular_cone c2 = new Circular_cone(2, 10);

    }

    2、接口

    接口只有被类实现后才能创建对象。

    接口中只能声明常量。public abstract 可以省略。

    接口的继承和类继承相同。一个接口可以继承自多个接口。

    一个类可以实现多个接口，用implements引导，用‘，’分割。可在实现接口的同时继承超类，继承在前。

    覆盖方法时，public不能省略。接口中的变量可覆盖，也可不覆盖。

    类引用接口要实现他的全部方法。

    //Controller.java
public interface Controller {
    public abstract void powerOnOff();
    public abstract void volumnUp(int increment);
    public abstract void volumnDown(int increment);
    public abstract void outPut();
}
//RemoteController.java
public interface RemoteController extends Controller{


}
//TV.java
public class Tv implements RemoteController{
    boolean power;
    int volumn;
    int volumnTemp;
    Tv(boolean power,int volumn)
    {
        this.power=power;
        this.volumn=volumn;
    }
    public void powerOnOff()
    {
        power=!power;
        if(power)
            volumn = volumnTemp;
        else{
            volumnTemp=volumn;
            volumn=0;
        }
    }
    public void volumnUp(int increment)
    {
        volumn=volumn+increment;
    }
    public void volumnDown(int increment)
    {
        volumn=volumn-increment;
    }
    public void outPut(){
        System.out.println("now the tv power "+power+",");
        System.out.println("now the tv value "+volumn);
    }
}
//Example2.java
public class Example2 {
    public static void main(String args[])
    {
        Tv t = new Tv(true,10);
        t.outPut();
    }
}

    2、异常捕获及处理

    从Throwable开始，Error和Exception继承于他，RuntimeException继承于Exception。

    Throwable是所有异常的超类。

    Error定义了系统级的底层错误，如内存耗尽，JVM崩溃，底层库调用错误等。

    Exception用于表示异常，表示程序有可能恢复的异常情况。

    RuntimeException是运行时异常。如NumberFormatException数字格式化异常。

    //Division.java
package cal;

public class Division {
    public static void d(String a,String b)
    {
        try{
            int x = Integer.parseInt(a);
            int y = Integer.parseInt(b);
            System.out.println(x/y);
        }catch(NumberFormatException e)
        {
            System.out.println("请输入数字！");
        }

    }
}

    5、自定义异常

    要求直接或者间接继承Exception类。

    继承Exception属于必须处理的异常。

    继承RuntimeException则不强制被处理。

    package cal;

public class Circle {
    public static void compute(String str) throws RadiusException
    {
        try{
            int r = Integer.parseInt(str);
            if(r<0){
                RadiusException e = new RadiusException();
                throw e;
            }
            System.out.println("圆的面积是："+2*Math.PI*r);
        }catch(NumberFormatException e){
            System.out.println("请输入数字！");
        }
    }
    static class RadiusException extends Exception{
        public String e_message(){
            return "出现异常：圆的半径不能小于0";
        }
    }
}

    3、UDP网络编程

    客户端编程步骤：建立连接-----发送数据-----接受数据-----关闭连接

    import java.io.InputStream;
import java.io.OutputStream;
import java.net.*;
import java.sql.Date;

public class App {
    public static void main(String[] args) throws Exception {
       DatagramSocket ds = null; //声明连接对象
       DatagramPacket sendDp=null; //声明发送数据包对象
       DatagramPacket receiveDp=null;//声明接收数据包对象
       try{
           //建立连接
           ds=new DatagramSocket();
           //初始化发送数据
           Date d = new Date(0);//获取当前时间
           String content = d.toString();
           byte[] data = content.getBytes();
           //初始化发送包对象
           InetAddress address = InetAddress.getByName("localhost");
           sendDp=new DatagramPacket(data, data.length,address,8880);
           //发送数据
           ds.send(sendDp);
           //初始化接受数据
           byte[] b = new byte[1024]; 
           receiveDp = new DatagramPacket(b, b.length);
           //接受数据
           ds.receive(receiveDp);
           //读取反馈内容，并输出
           byte[] response=receiveDp.getData();
           int len = receiveDp.getLength();
           String s = new String(response,0,len);
           System.out.println("服务器端反馈为："+s);

    服务器端编程步骤：建立一个连接-----接收客户端发送过来的数据-----对从客户端接收到的数据进行逻辑处理然后将处理结果发送会客户端-----关闭服务器端连接

    import java.io.InputStream;
import java.io.OutputStream;
import java.net.*;
import java.sql.Date;

public class App {
    public static void main(String[] args) throws Exception {
       DatagramSocket ds = null; //声明连接对象
       DatagramPacket sendDp=null; //声明发送数据包对象
       DatagramPacket receiveDp=null;//声明接收数据包对象
       String response = "hello java";
       try{
           //监听端口8880
           ds=new DatagramSocket(8880);
           System.out.println("服务器端已启动，等待客户端连接...");
           //初始化接受数据
           byte[] b = new byte[1024]; 
           receiveDp = new DatagramPacket(b, b.length);
           //接受数据
           ds.receive(receiveDp);
           //读取反馈内容，并输出
           InetAddress clientIP = receiveDp.getAddress(); 
           int clientPort = receiveDp.getPort();
            byte[] data = receiveDp.getData();
            int len = receiveDp.getLength();
            System.out.println("客户端IP："+clientIP.getHostAddress());
            System.out.println("客户端端口："+clientPort);
            System.out.println("客户端发送内容："+new String(data ,0, len));

            //发送反馈
            byte[] bData = response.getBytes();
            sendDp = new DatagramPacket(bData, bData.length,clientIP,clientPort);
            //发送数据
            ds.send(sendDp);
       }catch(Exception e){
            e.printStackTrace();
       }finally{
           try{
               ds.close();
           }catch(Exception e){
                e.printStackTrace();
           }
       }
    }
}
